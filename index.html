Руководство по настройке маршрутизатора ESR-1000

Базовые настройки

Для первоначальной настройки маршрутизатора Eltex ESR-1000 выполните следующие шаги:

  1. Подключите маршрутизатор к сети питания
  2. Подключите компьютер к порту ETH0 с помощью Ethernet-кабеля
  3. Откройте веб-браузер и перейдите по адресу 192.168.1.1
  4. Используйте стандартные учетные данные: admin/admin
  5. Смените пароль администратора в разделе System > User Management

Настройка WAN-подключения

Для настройки подключения к интернету:

// Пример настройки PPPoE подключения configure terminal interface GigabitEthernet0/1 no shutdown pppoe enable pppoe-client dial-pool-number 1 exit interface Dialer1 ip address negotiated encapsulation ppp dialer pool 1 ppp chap hostname user@example.com ppp chap password 0 mypassword exit

Настройка Wi-Fi

Для настройки беспроводной сети:

  1. Перейдите в раздел Wireless > Basic Settings
  2. Включите беспроводное соединение
  3. Установите имя сети (SSID)
  4. Выберите режим безопасности WPA2-Personal
  5. Установите надежный пароль
  6. Сохраните настройки и перезагрузите устройство

Решение частых коорпаративных вопросов

Вопрос 1: Как настроить VLAN на коммутаторе Eltex LTP-8X?

Решение:

Для настройки VLAN на коммутаторе Eltex выполните следующие команды:

// Создание VLAN configure terminal vlan 10 name Office exit // Назначение портов VLAN interface range ethernet 0/1-4 switchport mode access switchport access vlan 10 exit

Для настройки транкового порта:

interface ethernet 0/8 switchport mode trunk switchport trunk allowed vlan 10,20,30 exit

Вопрос 2: Как настроить резервирование DHCP на устройствах Eltex?

Решение:

Для настройки резервирования DHCP выполните следующие шаги:

  1. Определите MAC-адрес устройства, для которого требуется резервирование
  2. Откройте веб-интерфейс маршрутизатора
  3. Перейдите в раздел Services > DHCP Server
  4. Нажмите "Add" в разделе "Static Leases"
  5. Введите MAC-адрес устройства и желаемый IP-адрес
  6. Сохраните настройки

Пример настройки через CLI:

configure terminal ip dhcp pool STATIC-POOL host 192.168.1.50 255.255.255.0 hardware-address aa:bb:cc:dd:ee:ff client-name MyPC exit

Вопрос 3: Как настроить QoS для VoIP трафика?

Решение:

Для приоритизации VoIP трафика на маршрутизаторе Eltex:

  1. Создайте класс трафика для VoIP
  2. Настройте политику качества обслуживания
  3. Примените политику к интерфейсу
// Создание ACL для VoIP трафика ip access-list extended VOIP-ACL permit udp any any range 16384 32767 exit // Создание класса трафика class-map match-any VOIP-CLASS match access-group name VOIP-ACL exit // Создание политики QoS policy-map VOIP-POLICY class VOIP-CLASS priority percent 20 class class-default fair-queue exit // Применение политики к интерфейсу interface GigabitEthernet0/0 service-policy output VOIP-POLICY exit

Вопрос 4: Как настроить isp для получения трафика?

нужные команды:

Для получения трафика трафика на маршрутизаторе Eltex:

hostnamectl hostname isp
exec bash
apt-get update
apt-get install mc
cd /etc/net/ifaces
ls
cp -r ens18 /etc/net/ifaces/ens19
cp -r ens18 /etc/net/ifaces/ens20
cd ens19
cedit options
BOOTPROTO=static
F2 - Save F10 - exit
echo 172.16.4.1/28 > ipv4address
cd /etc/net/ifaces/ens20
mcedit options
BOOTPROTO=static
F2 - Save F10 - exit
echo 172.16.5.1/28 > ipv4address
cd
apt-get -y install firewalld
systemctl enable --now firewalld
firewall-cmd --permanent --zone=public --add-interface=ens18
firewall-cmd --permanent --zone=trusted --add-interface=ens19
firewall-cmd --permanent --zone=trusted --add-interface=ens20
firewall-cmd --permanent --zone=public --add-masquerade
firewall-cmd --complete-reload
systemctl restart network
ping ya.ru
ping ya.ru -I 172.16.4.1
ping ya.ru -I 172.16.5.1
